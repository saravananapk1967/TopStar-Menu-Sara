# TopStar Restaurant Menu Website

TopStar Restaurant Menu Website is a simple and attractive restaurant menu website developed using **HTML** and **basic CSS**. The website showcases a variety of food items with images, descriptions, and ordering options in a clean and user-friendly layout.

This project was created to practice and improve front-end web development skills, focusing on HTML structure, CSS styling, layout design, and responsive presentation of menu items.

## Features

* Restaurant menu display
* Food images and descriptions
* Order Now buttons
* Clean and simple user interface
* Built using HTML and CSS

## Technologies Used

* HTML
* CSS
